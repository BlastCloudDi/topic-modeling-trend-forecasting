# Прогнозирование смысловых трендов в стратегическом дискурсе

Программный модуль для автоматизированного выявления и прогнозирования динамики тематических трендов на основе методов тематического моделирования (BERTopic) и анализа временных рядов (SARIMA, CatBoost, LSTM).

## Источники данных

Корпус текстов сформирован на основе сообщений публичных Telegram-каналов за период с ноября 2016 по декабрь 2025 года:

- [Агентство стратегических инициатив (АСИ)](https://t.me/ASI_RU)
- [Настоящее будущее (НТИ)](https://t.me/nti2035ru)
- [Сильные идеи для нового времени](https://t.me/ideas_forum)

Общий объем корпуса: около 9000 сообщений.

## Архитектура проекта

Проект разделён на два этапа:

### Исследовательский этап (Google Colab)

Выполняется однократно. Включает полный цикл обработки:

1. **Сбор данных** - парсинг JSON-файлов Telegram.
2. **Предобработка** - лемматизация (`spaCy`), удаление стоп-слов и эмодзи.
3. **Тематическое моделирование** - BERTopic (UMAP + HDBSCAN + c-TF-IDF).
4. **Анализ временных рядов** - STL-декомпозиция, классификация тем.
5. **Прогнозирование** - Seasonal Naive, SARIMA, CatBoost, LSTM.
6. **Сохранение моделей** - экспорт в файлы для дашборда.

Итоговый ноутбук: `colab_pipeline.ipynb`

### Демонстрационный этап (Streamlit)

Интерактивный веб-дашборд, загружающий предобученные модели:

- Выбор темы, горизонта прогноза и модели.
- Визуализация исторической динамики и прогнозов.
- Сравнение четырёх моделей на одном графике.
- Просмотр ключевых слов темы и метрик качества.

Код дашборда: `app.py`


## Установка и запуск

### Требования

- Python 3.10+
- 4 ГБ оперативной памяти
- 500 МБ свободного места на диске

### Установка

```bash
git clone https://github.com/BlastCloudDi/topic-modeling-trend-forecasting.git
cd topic-modeling-trend-forecasting
pip install -r requirements.txt
```

### Запуск дашборда

```bash
streamlit run app.py
```
